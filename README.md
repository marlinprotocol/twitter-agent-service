# oyster-x-agent

AI agent that automates the process of sending tweets from an X account managed by Oyster.

## .env file
```
OPENAI_API_KEY=
USER_PASSWORD=
USERNAME=
USER_EMAIL=
USER_EMAIL_PASSWORD=
KMS_GENERATED_PASSWORD=
```
